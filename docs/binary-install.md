# CLI Installation Guide

To run this playground, you need the following tools installed on your machine.

## 1. Docker
Required to run the Kind cluster.
- [Install Docker Desktop](https://www.docker.com/products/docker-desktop/)

## 2. Kind
Creates the Kubernetes cluster inside Docker.
- [Installation Guide](https://kind.sigs.k8s.io/docs/user/quick-start/#installation)

## 3. kubectl
The command line tool for interacting with Kubernetes.
- [Installation Guide](https://kubernetes.io/docs/tasks/tools/#kubectl)

## 4. Git
Required for version control.
- [Installation Guide](https://git-scm.com/install/)
